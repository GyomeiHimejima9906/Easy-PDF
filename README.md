# Easy-PDF
A personal vibe coding project for easily managing PDFs.
I mad this for easily manage PDFs

# install
1. install node.js
2. run following commandss:
```
npm install electron --save-dev
npm install electron-builder --save-dev
```
3. install dipendency with
```
npm install --force
npm audit fix
npm install electron --save-dev
npm install electron-builder --save-dev
```
or by running the REQUIREMENTS.bat

4. run with
```
npm run electron:build
```
for windows use START.bat
